API

Endpoints

  • GET /stats - statistics

  • GET /stats/timedout - modules that timed out during verification

  • GET /stats/failed - failed to verify module's test existence

  • GET /stats/ok - modules with passed tests

  • GET /stats/nok - modules with failed tests

  • GET /stats/withouttests - modules without tests

  • GET /info/[module] - get test info (test status, output, date)
